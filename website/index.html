Miaoda - Claude Code Manager

一站式 Claude Code 与 OpenAI 兼容代理配置管理器。内置代理转换、模型映射、错误处理与可视化界面。

GitHub 仓库

主要功能

OpenAI 兼容代理

将 Claude /v1/messages 请求转换为 OpenAI /v1/chat/completions，支持 OpenAI、Azure OpenAI、Ollama 等。

模型映射

基于模型名称包含关键词进行映射：haiku → SMALL_MODEL、sonnet → MIDDLE_MODEL、opus → BIG_MODEL。

一键示例

在界面中一键填充 OpenAI / Azure / Ollama 示例，快速完成配置。

内置代理与安全

可自定义 Host/Port/Timeout，支持设置预期的 ANTHROPIC_API_KEY 进行客户端访问校验。

快速开始

  1. 打开应用 → 配置管理
  2. 勾选“启用内部代理”、“启用 OpenAI 模式”
  3. 填写 OPENAI_BASE_URL 与 OPENAI_API_KEY
  4. 根据需要设置 SMALL/MIDDLE/BIG_MODEL（也可点击一键示例）
  5. 保存并启动代理，测试连接

示例配置

OpenAI

OPENAI_BASE_URL=https://api.openai.com/v1
OPENAI_API_KEY=sk-...
SMALL_MODEL=gpt-4o-mini
MIDDLE_MODEL=gpt-4o
BIG_MODEL=gpt-4o

Azure OpenAI

OPENAI_BASE_URL=https://<资源名>.openai.azure.com/openai/deployments/<部署名>
OPENAI_API_KEY=<azure-key>
SMALL_MODEL=gpt-35-turbo
MIDDLE_MODEL=gpt-4
BIG_MODEL=gpt-4

Ollama

OPENAI_BASE_URL=http://localhost:11434/v1
OPENAI_API_KEY=dummy-key
SMALL_MODEL=llama3.1:8b
MIDDLE_MODEL=llama3.1:70b
BIG_MODEL=llama3.1:70b
